G마켓 리뉴얼페이지(반응형)

Website Figma Source

프로젝트 주제 :
방문자 순위 탑 사이트 리뉴얼 프로젝트(3대쇼핑몰) 선정 메인 페이지 리뉴얼
프로젝트 기간 :
2025.03.26 ~ 2025 04.25
프로젝트 소개 :
국내 3대 쇼핑몰 중 하나인 G마켓 사이트를 반응형으로 리뉴얼한 팀 프로젝트입니다. 실제 사용자 흐름과 UI/UX 패턴을 분석하여 메인 페이지, 회원가입, 고객센터, 상품상세 페이지, 장바구니 등 핵심 기능을 중심으로 구현하였습니다. 쇼핑몰의 다양한 기능을 웹 표준 기반으로 재구성하였고, PHP와 MySQL을 활용해 동적인 데이터 처리 및 회원 관리, 상품문의 기능을 직접 구현하였습니다. 탭 UI, 세션 기반 로그인 처리 등의 기능을 포함하고 있으며, JQuery로 인터랙션을 강화했습니다. 실제 서비스에 가까운 구조와 흐름을 반영해, 풀스택 웹 프로젝트의 구조와 흐름을 체계적으로 연습한 결과물입니다.
맡은역할 :
1. 로그인 페이지 개발 (폼 유효성 포함)
2. 전체 UI 스타일링 (Figma 기반 디자인 반영)
3. 상품 상세 페이지(item.php) 개발(옵션 선택, 총 가격 계산 로직 구현)
기술스펙 :
FIGMA, HTML5, CSS3, JAVASCRIPT, JQUERY, PHP
프로젝트 배포 :
Frontend - 닷홈 / DB - phpAdmin(MY-SQL)

Website Figma Source

중고매매 사이트 구현

2025.05.15 ~ 2025 06.20
프로젝트 소개 :
중고물품을 등록하고 거래할 수 있는 중고매매 플랫폼을 반응형으로 구현한 팀 프로젝트입니다. 사용자 로그인/회원가입, 상품 등록 및 수정, 장바구니 등 중고거래 사이트의 주요 기능을 직접 개발하였습니다. React 기반 컴포넌트 구조로 UI를 설계하고, 상태 관리와 라우팅, 유저 세션 등을 통해 실제 서비스 흐름과 유사하게 구현하였습니다. SASS를 활용해 모듈화된 스타일링을 구성하였으며, Figma를 통해 전체 UI/UX 설계와 디자인을 직접 진행하였습니다. 클라우드 타입을 통한 백엔드 서버 구축 및 AWS EC2 배포 환경을 경험하였고, MySQL과 연동된 데이터 관리 기능도 포함되어 있습니다.
맡은역할 :
1. 네비게이션 메뉴 클릭시 연결되는 sub page 개발
2. 전체 UI 스타일링 (SASS 모듈화 및 Figma 기반 디자인 반영)
3. 장바구니 상태관리 및 로컬스토리지 연동(상품 상세 페이지에서 장바구니 연동, 삭제, 가격 계산 로직 구현)
4. 상품 등록/수정(write.js, edit.js) 페이지 및 이미지 미리보기 기능 구현
기술스펙 :
FIGMA, HTML5, SASS, REACTP
프로젝트 배포 :
Frontend - 깃허브 / back - 클라우드 타입 / DB - phpAdmin(MY-SQL)
